# Задание 2: Анализатор текста (Коллекции, Владение)

**Цель:** Практика работы с `String` и `HashSet`.

**Задача:** Напишите функцию `analyze_text(text: &str) -> (usize, usize, String)`.

**Требования:**

Функция должна возвращать кортеж из:
1. Количества слов (разделитель - пробел);
2. Количества уникальных слов (регистр не учитывать);
3. Самое длинное слово (если их несколько, то возвращать первое).

Пример:
```rust
let text = "hello world hello again";
assert_eq!(analyze_text(text), (4, 3, "hello".to_string()));
```
